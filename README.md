## 🧮 Key Formulas Used

| Formula | Expression |
|---------|------------|
| Basic Probability | P(E) = Favourable ÷ Total |
| Addition Rule | P(A or B) = P(A) + P(B) - P(A and B) |
| Conditional Probability | P(A given B) = P(A and B) ÷ P(B) |
| Sampling Error | Sample Mean - Population Mean |
| Standard Error of Mean | SEM = σ ÷ √n |
| Independent Events | P(A and B) = P(A) × P(B) |
